# SkillyHire Job Portal

A full-stack job portal built using React and Node.js.

## 🔹 Tech Stack

* Frontend: React
* Backend: Node.js / Express
* Database: PostgreSQL (Neon)

## 🔹 Features

* User authentication
* Job posting
* Job application system

## 🔹 Deployment

* Frontend: Vercel
* Backend: Serverless (Vercel)
